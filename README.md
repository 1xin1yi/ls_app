# ls_app

安装教程

1.安装Wampserver64

2.安装Navicat Premium 15

3.安装vs code

使用说明

1.启动Wampserver64，将数据库的数据导入Navicat Premium 15中，将apiLearn文件夹在vs code打开，在终端运行nodemon app.js。

2.将ls_graduation_design文件夹在vs code打开，在终端运行npm run dev。
